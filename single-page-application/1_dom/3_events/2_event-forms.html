Forms

GlobalEventHandlers.onfocus;

Das event .onfocus; feuert ein callback wenn ein element in den focus gerät, also zum beispiel ein input element das angeklickt wurde.

  • element.addEventListener('focus', (event) => {});
  • element.onfocus = (event) => {};

GlobalEventHandlers.onblur;

Dieses event wird ausgelöst wenn wir den focus von einem element verlieren, also zum beispiel wenn wir ein input feld verlassen.

  • element.addEventListener('blur', (event) => {});
  • element.onblur = (event) => {};

GlobalEventHandlers.onsubmit;

Das onsubmit event wird abgefeuert wenn ein formular gesehndet, also submitted wird.

  • element.addEventListener('submit', (event) => {});
  • element.onsubmit = (event) => {};

Das formular wurde abgeschickt!

GlobalEventHandlers.onchange;

onchange wird immer dann abgefeuert, wenn die werte ines inputs vom benutzer verändert werden, zum beispiel wenn der benutzer in ein textfeld schreibt.

  • element.addEventListener('change', (event) => {});
  • element.onchange = (event) => {};

GlobalEventHandlers.onclick;

Das event wir abgefeuert wenn der benutzer auf eine element klickt.

Ein klick wird ausgeführt, nachdem die maustaste im selben bereich sowohl runter gedrückt, wie auch losgelassen wurde.
  • element.addEventListener('click', (event) => {});
  • element.onclick = (event) => {};
