Mouse

GlobalEventHandlers.onmousemove;

Dieses event wird abgefeuert wenn der benutzer die maus innerhalb eines selektierten elements bewegt. Das kann das ganze dokument oder auch einzelne elemente sein.

  • element.addEventListener('mousemove', (event) => {});
  • element.onmousemove = (event) => {};

GlobalEventHandlers.onmouseover;

DIeses event wird abgefeuert wenn der benutzer die maus über ein element bewegt. Der unterschied zwischen .onmousemove; und .onmouseover; ist das .onmouseover; nur einmal abgefeuert wird, wenn das element "gehovert" wird, während es bei .onmousemove; konstanzt abgefeuert wird.

  • element.addEventListener('mouseover', (event) => {});
  • element.onmouseover = (event) => {};

GlobalEventHandlers.onmouseout;

Dieses event wird abgefeuert wenn wir mit der maus ein element verlassen.

  • element.addEventListener('mouseout', (event) => {});
  • element.onmouseout = (event) => {};

GlobalEventHandlers.onmousedown;

Diesen event wird ausgelöst wenn wir eine maustaste runterdrücken.

  • element.addEventListener('mousedown', (event) => {});
  • element.onmousedown = (event) => {};

GlobalEventHandlers.onmouseup;

Dieses event wird abgefeuert wenn wir eine maustaste loslassen.

das event .onclick wird abgefeuert nachdem .onmousedown und .onmouseup auf dem selben element ausgeführt wird.
element -> .onmousedown -> .onmouseup = .onclick
  • element.addEventListener('mouseup', (event) => {});
  • element.onmouseup = (event) => {};
