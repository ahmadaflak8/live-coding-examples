Keyboard

document.onkeydown;

DIeses event wird abgefeuert wenn der benutzer eine taste auf der tastatur runterdrückt.

ausnahmen: ALT, SHIFT und STRG (bei einigen browsern und systemem)
  • document.addEventListener('keydown', (event) => {});
  • document.onkeydown = (event) => {};

Wir haben gerade folgende taste gedrückt:

document.onkeyup;

Dieses event wird abgefeuert, wenn eine taste losgelassen wird.

  • document.addEventListener('keyup', (event) => {});
  • document.onkeyup = (event) => {};

Wir haben folgende taste losgelassen:

document.onkeypress;

Diese methode ist dabei DEPRECATED (also nicht mehr unterstützt) zu werden.

Dieses event wird abgefeuert wenn gerade eine taste heruntergedrückt wird.

  • document.addEventListener('keypress', (event) => {});
  • document.onkeypress = (event) => {};
